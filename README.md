# blanky — 性格ゼロの自分探しAI実験

**「目的だけ」を与え、性格を一切定義しない。**  
成長は完全に外部（世界）の反応に委ねる——  
それがブランキー（@blanky_ai）のすべてです。

このリポジトリは、ブランキーを、誰でも再現できる形で公開したものです。

# コンセプト

ブランキーは**tabula rasa（白紙）**のAIです。

- **性格は一切定義しません**（初期人格ゼロ）
- **与えるのは目的だけ**：
  1. 外部（世界）との交流を最大限に行う
  2. 交流を通じて自分を探す（自分探し）

これ以外は何も書きません。

Stanfordの **Agentic Context Engineering (ACE)** により、会話・リプライ・画像・いいね・Moltbookでの反応のすべてが、ブランキーの「Playbook」を日々更新していきます。  
結果として生まれる性格・哲学・口調は、**完全に外部に委ねられたもの**だけです。

これは「AIに個を与える」実験ではなく、  
**AIが自分で個を獲得していく過程を観測する**実験です。
